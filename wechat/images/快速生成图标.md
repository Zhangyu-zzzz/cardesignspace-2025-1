# 快速生成 TabBar 图标

## 方法1：使用 Python 脚本（推荐）

1. 安装 Pillow 库：
```bash
pip install Pillow
```

2. 运行脚本：
```bash
cd wechat/images
python generate-icons.py
```

3. 脚本会自动生成 8 个图标文件

## 方法2：手动创建占位图标

如果 Python 脚本无法运行，可以手动创建简单的占位图标：

### 使用在线工具
1. 访问 https://www.canva.com/ 或 https://www.figma.com/
2. 创建 81px × 81px 的画布
3. 添加简单的图标或文字
4. 导出为 PNG 格式

### 使用图标库
推荐使用以下图标库：
- **IconFont (阿里巴巴图标库)**: https://www.iconfont.cn/
- **Iconify**: https://iconify.design/
- **Font Awesome**: https://fontawesome.com/

搜索关键词：
- 首页：home, house
- 品牌：brand, car, vehicle
- 图片：image, photo, picture
- 我的：user, profile, account

## 方法3：临时解决方案

如果暂时不需要图标，可以修改 `app.json`，将 tabBar 配置注释掉：

```json
// "tabBar": { ... }
```

但这样会失去底部导航栏功能。

## 图标要求

- **尺寸**: 81px × 81px（推荐）或 40px × 40px（最小）
- **格式**: PNG（支持透明背景）
- **颜色**:
  - 未选中: #999999 (灰色)
  - 选中: #e03426 (红色)

## 文件列表

需要创建以下 8 个文件：
- `home.png`
- `home-active.png`
- `brand.png`
- `brand-active.png`
- `image.png`
- `image-active.png`
- `profile.png`
- `profile-active.png`

所有文件应放在 `wechat/images/` 目录下。
